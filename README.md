# 프로젝트 개요
| 구분 | 내용 |
|:---|:---|
| **주제** | 형성평가 및 복습을 위한 **실시간 웹 행맨 게임** |
| **목표** | 기존의 지루한 평가 방식에서 벗어나, **문제 기반**의 퀴즈 형식과 **실시간 경쟁** 요소를 도입하여 학습에 대한 재미와 동기 부여 증진 |
| **차별점**| 1. **문제 제시**: 정답 단어와 관련된 문제가 주어져 맥락적 학습 유도 <br> 2. **실시간 멀티플레이**: WebScoket을 활용하여 여러 사용자가 동시에 참여하고, 채팅 및 게임 상태를 즉시 공유하며 경쟁 |
| **주요 기능** | 실시간 채팅, 게임 화면 동기화, 한글 자모/단어 입력 및 검증, 행맨 그래픽 업데이트 |

<br>

# 기술 설명 및 스택
### 1. 기술 스택
| 분야 | 주요 기술 | 설명 |
|:---|:---|:---|
|**프론트엔드**| HTML5, CSS3, JavaScript | 별도의 프레임워크 없이 표준 기술만을 사용하여 경량화된 UI/UX 구현
|**백엔드**| Java | 안정적이고 빠른 처리를 위해 Java 기반으로 서버 로직 구현 |

<br>

### 2. 핵심 로직 상세 설명
**2.1. WebSocket기반 실시간 통신**:
- 메시징 및 게임 이벤트(자모 입력, 정답 여부, 행맨 상태를 **실시간**으로 브로드캐스팅하여 모든 참가자의 화면을 동기화합니다.

<br>

**2.2. 한글 유니코드 기반 자모 분리 및 검증:**
- 정답 단어를 유니코드로 변환 후, **초성, 중성, 종성** 단위로 분리하여 저장합니다.
- 사용자 입력이 한글 자모인지 검증하고, 분리된 자모와 매핑하여 정답 처리의 정확도를 높였습니다.
- 사용자가 **한 글자를 입력하면** 해당 문자가 자모인지 확인합니다.
- **두 글자 이상 입력하면**, 입력된 단어가 정답과 일치하는지 확인합니다.
